# Shaders and Shading Language

## Shaders

A shader is a program, which originally computed the shading of 3D scenes -
light, darkness and color - but can be also used for special effects, post processing and even computations that have to relation to computer graphics.

Shaders are implemeted in a _shading language_, a programming language to
express the computations to perform. The shading language depends heavily
on the target environment, making it difficult to express concepts across
different environments. While MGE allows to directly use the environment's
language, it provides an abstracted language to ease development of shaders.
