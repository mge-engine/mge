Shader Programs¶

A shader program is an user-defined program which is executed as part of a render pipeline on a graphics card.

Types and Definitions¶

Shader Types¶

enum mge::shader_type¶

Type of shader.

Values:

enumerator VERTEX¶

vertex

enumerator FRAGMENT¶

fragment (pixel)

enumerator COMPUTE¶

compute

enumerator TESSELATION_CONTROL¶

tesselation control (hull)

enumerator TESSELATION_EVALUATION¶

tesselation evaluation (domain)

enumerator GEOMETRY¶

geometry

enumerator MAX_SHADER_TYPE¶

Shader¶
