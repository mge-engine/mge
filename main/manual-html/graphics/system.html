Render System

The render system is the basic entry point into the graphics library.

The default render system can be selected by

  • setting the environment variable MGE_RENDER_SYSTEM, or

  • setting the configuration parameter graphics.render_system.

MGE_RENDER_SYSTEM

Specifies the render system implementation to be used, such as “opengl” or “vulkan”.

class render_system : public mge::component<render_system>

Render system interface.

The render system defines a mechanism to produce graphics output, that is:

  • open and manage windows to present graphics output

  • manage the capabilities of the render system

  • create and manage the objects of a graphics pipeline, such as shaders and buffers
