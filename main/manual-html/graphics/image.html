Image

An image provides an interface to the data and metadata of a 2D pixel image.

A ‘memory_image’ is an image that owns its own memory.

Types and Definitions

class image

An image. Base class for image types.

An image has an extent (size) and a format. Images are immutable, i.e. format and size cannot change inplace

Subclassed by mge::memory_image
