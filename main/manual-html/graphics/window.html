Window¶

To present something on a screen, a window has to be created. The window can of course occupy the whole screen.

class window : public std::enable_shared_from_this<window>, public mge::input_handler¶

A window.

Public Types

using close_listener = void_function¶

Listener type for close action.

using closing_listener = std::function<bool()>¶

Listener to check on closing action.

This listener will be called on close action. If the listener returns false, the window will not be closed.
