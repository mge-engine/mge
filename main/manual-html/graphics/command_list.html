Command Lists¶

Command lists capture state changes on a graphics pipeline, that is, setting buffers and shaders, and submitting draw calls that make use of them.

Command lists can be native, i.e. implemented by the underlying render API.

class mge::command_list : public mge::context_object¶

A sequence of state changes.

A list of state-changing commands which can be applied to a render context.

Public Functions

virtual ~command_list()¶

Destructor.

bool native() const¶

Returns whether this command list is native.

Some render APIs support command list objects on their API level. If the command list is mapped to such an object, a command list is native.

Returns

true if native
