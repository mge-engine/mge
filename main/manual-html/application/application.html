Application

The application class provides facilities to create a typical MGE application.

class application : public mge::component<application>, public mge::loop_target

Application framework.

An application allows to attach input, update, and redraw listeners which are called during the update loop.

Also a quit listener can be attached which is called when the application is terminating.

virtual void update(uint64_t cycle, double elapsed, double delta) override

Update state in a step.

Note cycle may be the same for multiple calls.

Parameters:

  • delta – delta in seconds

virtual void present(uint64_t cycle, double peek) override

Parameters:
