Mouse Handling¶

Constants for mouse actions are defined in the mge::mouse_action enumeration.

enum class mge::mouse_action¶

Codes for mouse actions.

enumerator PRESS¶

Button press.

enumerator RELEASE¶

Button release.

enumerator DOUBLE_CLICK¶

Double click, as reported by operating system.

enumerator MOUSE_ACTION_DOUBLE_CLICK¶

Alias for DOUBLE_CLICK, which may be defined as preprocessor symbol.
