Concurrency¶

MGE uses multi-threading to achieve concurrency.

Threading Utilities¶

Thread¶

The thread class enhances the std::thread interface by some convenience methods.

class mge::thread : private mge::noncopyable, public std::enable_shared_from_this<thread>¶

A thread.

This class enhances std::thread by some convenience methods. A thread cannot be copied, or created on the stack.

Thread Group¶

Threads can be organized in thread groups. A thread is assigned to a thread group om construction, and the group cannot be changed.

class thread_group : public mge::noncopyable, public std::enable_shared_from_this<thread_group>¶

A thread group.

A thread group is a collection of threads. Threads are added to the group by specifying the thread group at thread construction time, a thread cannot change its group after being assigned to one.
