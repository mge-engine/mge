Memory Management

MGE integrates the _jemalloc_ library for general purpose memory management.

For C-style memory management, the following functions are provided:

void *mge::malloc(size_t size)

Allocate memory.

An :any:mge::out_of_memory exception is thrown in case memory cannot be allocated.

Parameters:

size – number of bytes to allocate

Returns:

pointer to allocated memory

void *mge::realloc(void *ptr, size_t size)

Reallocate memory.

This function is similar to :any:realloc from the C standard library. If the pointer is :any:nullptr, the function behaves like :any:malloc. If the size is 0, the function behaves like :any:free.

Parameters:

  • ptr – pointer to memory to reallocate

  • size – new size of memory block

Returns:

pointer to reallocated memory

void mge::free(void *ptr)

Frees allocated memory.

Passing a :any:nullptr to the function is safe, no operation is performed in this case.

Parameters:

ptr – pointer to memory to free.

For C++-style memory management, a std::pmr::memory_resource is provided:

class memory_resource : public std::pmr::memory_resource

Memory resource. This memory resource uses the fast allocator provided with mge.

Note that you should not create instances of this memory resource, but rather use the static instance provided.

Public Static Attributes

static ::mge::memory_resource instance

Instance of the memory resource.

static std::pmr::polymorphic_allocator<void> allocator{get_memory_resource_instance()}

Allocator using the memory resource.
