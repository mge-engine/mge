Streams¶

MGE defines an input and outüut stream base, which allows an easy implementation of custom data sources or targets. These streams also each provide a method to retrieve a std::istream or std::ostream.

Input Stream¶

class mge::input_stream¶

An input stream to read bytes.

Public Types

enum direction_type¶

Seek direction type.

Values:

enumerator POS_BEG¶

seek from beginning

enumerator POS_CUR¶

seek from current position

enumerator POS_END¶

seek from end

using offset_type = std::streamoff¶

Type of positions and offsets.

using streamsize_type = std::streamsize¶

Type of number of bytes read.

Public Functions

input_stream() = default¶

Constructor.

virtual ~input_stream() = default¶

Destructor.

virtual streamsize_type read()¶

Read one byte from the input stream. The default implementation uses the read(void*,int) method to obtain the data.

Throws

IOException – if an I/O error occurs.

Returns

byte that has been read, or -1 at EOF

streamsize_type read(void *destination, streamsize_type size)¶

Read more bytes from the input stream. The default implementation is always at EOF.

Parameters

  • destination – target buffer

  • size – target buffer size

Returns

number of bytes read, 0 on EOF

virtual offset_type position()¶

Get current position.

Returns

current position, -1 if not known

virtual offset_type seek(offset_type offset, direction_type dir)¶

Seek relative to a specific position.

Parameters

  • offset – seek offset

  • dir – seek direction

Returns

position after seek, -1 if not known

std::istream &istream()¶

Access the stream as an std::istream

Returns

a std::istream using this stream’s data

bool eof()¶

Return whether stream is at end-of-file.

Returns

true if stream is at eof

l – listener

Returns

listener key

void remove_read_listener(unsigned int key)¶

Parameters

key – listener key
