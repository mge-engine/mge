Command Line and User Input

MGE provides utilities for parsing command-line arguments and handling interactive console input.

Program Options

The mge::program_options class provides a flexible framework for defining and parsing command-line options. It supports named options, positional arguments, and option values with type safety.

Line Editor

The mge::line_editor class provides interactive line input with prompt support for console applications. Only one line editor instance can exist at a time.

class line_editor

A line editor.

A line editor handles the input in the console. Only one line editor can exist at the same time.
