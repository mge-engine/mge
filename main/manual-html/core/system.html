System Abstractions¶

Some abstractions of common system functionality is provided, if that abstraction is not already covered by the standard library.

System Error¶

The system error captures the last - or a provided - system error and also resolves its message.

class mge::system_error : public mge::exception¶

A system error.

Constructing a system error retrieves the last system error.

struct error_code : public mge::exception::tag<error_code, error_code_type>¶
