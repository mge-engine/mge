**************
Mouse Handling
**************

Constants for mouse actions are defined in the :any:`mge::mouse_action` enumeration.

..  cpp:enum-class:: mge::mouse_action

    Codes for mouse actions.

    ..  cpp:enumerator:: PRESS

        Button press.

    ..  cpp:enumerator:: RELEASE

        Button release.

    ..  cpp:enumerator:: DOUBLE_CLICK

        Double click, as reported by operating system.

    ..  cpp:enumerator:: MOUSE_ACTION_DOUBLE_CLICK

        Alias for :any:`DOUBLE_CLICK`, which may be defined as preprocessor symbol.
